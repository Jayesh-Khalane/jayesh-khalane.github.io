# Jayesh Khalane Portfolio

Static GitHub Pages portfolio for Jayesh Khalane.

## Structure

- `docs/index.html` - homepage
- `docs/pages/` - experience, projects, blog, and certifications
- `docs/css/style.css` - shared site styles
- `docs/js/main.js` - navigation, animation, and date behavior
- `docs/assets/` - images, certificates, and resume PDFs

## Local Preview

Open `docs/index.html` in a browser.
